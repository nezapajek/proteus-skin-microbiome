# AMPLICON SEQUENCE ANALYSIS

## DESCRIPTION
- Single end analysis 
- DNA Metabarcoding of Bacteria, Archaea (16S rRNA), Eukaryota (18S rRNA), Fungi (ITS2)
- Raw Data = Demultiplexed Illumina sequences (MiSeq) in .fastq format

### REQUIREMENTS
- latest Qiime2 ver.
